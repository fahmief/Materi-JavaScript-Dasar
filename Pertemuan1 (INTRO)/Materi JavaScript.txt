2. string

operator dalam javascript
1. aritmatika (+,-,*,/,%(sisa bagi)) -> () * / + -
2. penugasan (assigment) -> =
3. perbandingan
4. logika
5. string

6. kondisional ternary (membutuhkan 3 operant)

perbandingan perbedaan == atau ===
== kalau sama walaupun beda tipe data
10 == "10" -> true
=== kalau sama dan harus sama tipe datanya
10 === 10 -> true
10 === "10" -> false
